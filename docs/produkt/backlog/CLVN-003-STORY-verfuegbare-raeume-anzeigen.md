# Verfügbare Räume anzeigen

## User Story

Als INNOQ-Mitarbeiter möchte ich verfügbare Räume am ausgewählten Standort anzeigen, damit ich eine Übersicht über alle buchbaren Konferenzräume erhalte.

## Beschreibung

Nachdem ein Mitarbeiter einen Bürostandort ausgewählt hat, benötigt er einen Überblick über die dort vorhandenen Konferenzräume und deren Verfügbarkeit. Diese Übersicht ist die Grundlage für eine informierte Raumauswahl und stellt sicher, dass nur tatsächlich buchbare Räume in Betracht gezogen werden.

## Akzeptanzkriterien

- [ ] Es werden alle Konferenzräume des zuvor ausgewählten Standorts angezeigt
- [ ] Pro Raum sind die zentralen Eckdaten (Name, Kapazität) auf einen Blick erkennbar
- [ ] Die Verfügbarkeit der Räume ist aus der Übersicht ersichtlich
- [ ] Ist am Standort kein Raum verfügbar, wird ein verständlicher Hinweis angezeigt
- [ ] Von der Übersicht aus kann ein Raum für die weitere Betrachtung ausgewählt werden

## Definition of Done

- [ ] Code ist implementiert und per Pull Request reviewed
