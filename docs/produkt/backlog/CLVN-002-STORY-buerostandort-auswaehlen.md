# Bürostandort auswählen

## User Story

Als INNOQ-Mitarbeiter möchte ich einen Bürostandort auswählen, damit ich Räume an meinem gewünschten Standort finden kann.

## Beschreibung

INNOQ hat acht Standorte in Deutschland und der Schweiz. Bevor ein Mitarbeiter einen passenden Konferenzraum finden kann, muss er festlegen, an welchem Standort er einen Raum sucht. Die Standortauswahl ist der erste Schritt im Ablauf "Räume finden" und grenzt die anschließend angezeigten Räume auf den relevanten Standort ein.

## Akzeptanzkriterien

- [ ] Alle acht INNOQ-Standorte (Monheim, Berlin, Hamburg, Köln, München, Zürich, Baar, Offenbach) stehen zur Auswahl
- [ ] Es kann genau ein Standort ausgewählt werden
- [ ] Der ausgewählte Standort ist deutlich erkennbar
- [ ] Nach der Auswahl werden die Folgeschritte auf den gewählten Standort bezogen
- [ ] Die Auswahl kann jederzeit auf einen anderen Standort geändert werden
