# Räume nach Kapazität filtern

## User Story

Als INNOQ-Mitarbeiter möchte ich Räume nach Kapazität filtern, damit ich einen Raum finde, der groß genug für meine Teilnehmerzahl ist.

## Beschreibung

Meetings und Workshops haben unterschiedliche Teilnehmerzahlen. Damit ein Mitarbeiter nicht jeden Raum einzeln auf seine Größe prüfen muss, soll er die Raumübersicht nach Kapazität einschränken können. So werden nur Räume angezeigt, die für die geplante Teilnehmerzahl ausreichend Platz bieten.

## Akzeptanzkriterien

- [ ] Es kann eine benötigte Teilnehmerzahl bzw. Mindestkapazität angegeben werden
- [ ] Es werden nur Räume angezeigt, deren Kapazität die angegebene Teilnehmerzahl erfüllt oder übersteigt
- [ ] Der aktive Kapazitätsfilter ist erkennbar und kann zurückgesetzt werden
- [ ] Der Filter ist mit der Standortauswahl kombinierbar
- [ ] Liefert der Filter kein Ergebnis, wird ein verständlicher Hinweis angezeigt
