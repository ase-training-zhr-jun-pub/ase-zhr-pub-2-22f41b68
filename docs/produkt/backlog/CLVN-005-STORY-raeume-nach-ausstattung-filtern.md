# Räume nach Ausstattung filtern

## User Story

Als INNOQ-Mitarbeiter möchte ich Räume nach Ausstattung filtern, damit ich einen Raum mit der benötigten technischen Ausstattung finde.

## Beschreibung

Je nach Anlass benötigt ein Mitarbeiter eine bestimmte technische oder räumliche Ausstattung, etwa einen Bildschirm, ein Whiteboard oder Videokonferenz-Equipment. Über einen Ausstattungsfilter kann er die Raumübersicht gezielt auf Räume einschränken, die seine Anforderungen erfüllen.

## Akzeptanzkriterien

- [ ] Es können eine oder mehrere Ausstattungsmerkmale als Filterkriterium ausgewählt werden
- [ ] Es werden nur Räume angezeigt, die alle ausgewählten Ausstattungsmerkmale aufweisen
- [ ] Der aktive Ausstattungsfilter ist erkennbar und kann zurückgesetzt werden
- [ ] Der Filter ist mit der Standortauswahl und dem Kapazitätsfilter kombinierbar
- [ ] Liefert der Filter kein Ergebnis, wird ein verständlicher Hinweis angezeigt

## Definition of Done

- [ ] Alle Akzeptanzkriterien sind erfüllt und überprüft
- [ ] Code ist implementiert und per Pull Request reviewed
