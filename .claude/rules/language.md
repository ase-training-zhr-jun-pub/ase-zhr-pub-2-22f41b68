# Language

Use **English** for all file names and all file content — source code, comments,
variable names, commit messages, documentation, and rules.

The only exception is translation/i18n resource files (e.g. `messages.de.json`),
which may contain the target language.

Docs under `docs/` that are still in German are legacy; translate them when you
touch them, but do not translate them proactively.
